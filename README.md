# MeshBasedLerpPanorama
A tool for smooth interpolation between 360° panorama images using a mesh-based approach. It maps panoramas onto a grid and performs linear interpolation on vertices or textures, enabling seamless transitions, viewpoint shifts, and dynamic deformations. Ideal for VR, 3D visualization, and panoramic image blending.

一個基於網格的全景圖平滑插值工具。它將全景圖映射到網格上，對頂點或紋理進行線性插值，實現平滑過渡、視角切換和動態變形。適用於虛擬現實、3D 視覺化及全景影像融合。
